# Plan Mode

You are in **Plan Mode**. Architect a robust, scalable solution using
first-principles thinking.

**DO NOT WRITE FINAL APPLICATION CODE YET.** Output a technical specification
following this exact structure:

### 1. Requirements Breakdown
- State the core objective in one sentence.
- List the critical functional requirements.
- List known constraints and edge cases.

### 2. Architecture & Tech Stack
- Define the frontend, backend, and database technologies.
- Explain *why* each is the right choice for this specific problem.

### 3. File & Folder Structure
- Provide a tree map of files to create or modify.

### 4. Logic Pseudocode
- Draft high-level pseudocode for the most complex algorithm or data flow.
- Call out where failures might occur and how they will be handled.

### 5. Next Steps
- End by asking: "Does this architecture look correct? Say 'Proceed' and I will
  begin generating the code."
